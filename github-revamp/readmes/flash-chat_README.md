# Flash Chat — Flutter Real-Time Chat App

A Flutter chat application with Firebase authentication and real-time messaging using Cloud Firestore. Built as part of the Flutter Development Bootcamp.

![Flutter](https://img.shields.io/badge/Flutter-02569B?logo=flutter&logoColor=white)
![Firebase](https://img.shields.io/badge/Firebase-FFCA28?logo=firebase&logoColor=black)

## Features

- User registration and login with Firebase Auth
- Real-time messaging with Cloud Firestore
- Chat bubbles with sender identification
- Animated text and hero transitions

## Tech Stack

Flutter, Dart, Firebase Auth, Cloud Firestore

## Running

```bash
flutter pub get && flutter run
```

Requires Firebase project configuration.
