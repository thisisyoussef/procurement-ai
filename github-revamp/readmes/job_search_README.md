# Job Search — Cross-Platform Job Search App

A Flutter mobile app for browsing job listings with search, filtering, favorites, and detailed job views.

![Flutter](https://img.shields.io/badge/Flutter-02569B?logo=flutter&logoColor=white)
![Dart](https://img.shields.io/badge/Dart-0175C2?logo=dart&logoColor=white)

## Features

- Job listing feed with search and filtering
- Detailed job view with company info, requirements, and application link
- Favorites/bookmarking for saved jobs
- Clean Material Design interface
- Cross-platform (iOS + Android)

## Tech Stack

Flutter, Dart

## Running

```bash
git clone https://github.com/thisisyoussef/job_search.git
cd job_search && flutter pub get && flutter run
```
