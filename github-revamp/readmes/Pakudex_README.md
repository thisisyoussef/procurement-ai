# Pakudex — Creature Collection System

A Java creature collection system inspired by Pokemon. Menu-driven CLI interface for adding, listing, evolving, and managing "Pakuri" creatures with species tracking.

![Java](https://img.shields.io/badge/Java-ED8B00?logo=openjdk&logoColor=white)

## Features

- Add and manage creatures with species attributes
- Menu-driven command-line interface
- Evolution mechanics
- Collection statistics
