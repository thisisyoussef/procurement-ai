# Clima — Flutter Weather App

A Flutter weather app that uses location services and a weather API to display current conditions and forecasts. Built as part of the Flutter Development Bootcamp.

![Flutter](https://img.shields.io/badge/Flutter-02569B?logo=flutter&logoColor=white)
![Dart](https://img.shields.io/badge/Dart-0175C2?logo=dart&logoColor=white)

## Features

- Location-based weather using device GPS
- City search for weather in any location
- Current temperature, conditions, and weather icons
- API integration with OpenWeatherMap

## Tech Stack

Flutter, Dart, Geolocator, OpenWeatherMap API

## Running

```bash
flutter pub get && flutter run
```

Requires an OpenWeatherMap API key.
