# Weather Website — Node.js Weather Lookup

A weather lookup website built with Node.js, Express, and Handlebars templating. Uses geocoding and forecast APIs to display weather for any location.

![Node.js](https://img.shields.io/badge/Node.js-339933?logo=nodedotjs&logoColor=white)
![Express](https://img.shields.io/badge/Express-000000?logo=express&logoColor=white)

## Features

- Location search with geocoding API
- Current weather and forecast display
- Server-side rendering with Handlebars

## Tech Stack

Node.js, Express, Handlebars, Weather API, Geocoding API

## Running

```bash
npm install && npm start
```
