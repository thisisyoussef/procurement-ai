# NutrientHelper — Nutrient Tracking Tool

A C++ nutrient tracking and analysis tool with graph-based data structures and JSON food data storage using RapidJSON.

![C++](https://img.shields.io/badge/C++-00599C?logo=cplusplus&logoColor=white)

## Features

- Nutrient data lookup and analysis
- Graph-based data structure for food relationships
- JSON data storage with RapidJSON
- User profile management

## Tech Stack

C++, RapidJSON
